# ⚡ DataSage

### From Data to Decisions—Instantly

DataSage is a smart Excel automation tool that converts raw data into meaningful insights, charts, and downloadable reports in seconds.

## 📌 Features

* 📊 Instant Reports
  Generate complete Excel reports automatically

* 🔄 Flexible Column Mapping
  Works with different Excel formats

* 📈 Built-in Charts
  Automatically creates charts inside Excel

* ⏱ Saves Hours of Work
  Eliminates manual reporting effort

* 📥 Downloadable Reports
  Get structured Excel output with one click

---

## 🖥️ How It Works

1. Upload your Excel file
2. Select (map) required columns
3. Get insights and charts instantly
4. Download your automated report
